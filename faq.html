FAQ

Q: What diff tools are supported?

A: Any external diff tool that accepts file paths via command line: WinMerge, Beyond Compare, KDiff3, etc.

Q: Does it support folders?

A: Yes, you can compare folders as well as files.

Q: Does it work with classic and modern context menus?

A: Yes, both are supported on Windows 10/11.

Q: Is my data private?

A: Yes. diff-ext does not collect or send any data outside your system.

Q: How do I clear the MRU list?

A: Right-click and use the "Clear MRU" command in the submenu.

Q: How to report a bug?

A: Open a GitHub issue (see Support page).
