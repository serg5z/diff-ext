diff-ext

diff-ext is a modern Windows context menu extension that adds powerful file comparison and MRU (Most Recently Used) management features directly into File Explorer.
Open source. Fast. No bloat.

Quick Download

Download latest release » or Get MSIX package from Windows App Store »

  • Supports Windows 10/11 (x64, x86, ARM64)
  • Configure your preffered diff tool via diff-ext-setup.exe
  • Compare files, manage MRU list

Why diff-ext?

  • Compare files & folders with your favorite diff tool, right from Explorer
  • Remember files for later comparison (MRU)
  • Modern & classic context menu support
  • Open source, BSD license
