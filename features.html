Features

  • Compare Files & Folders: Select any two files/folders and compare using your configured diff tool.
  • MRU List: Remember files for later comparison; access them from a submenu.
  • Submenu for MRU: See and manage your MRU files directly in the context menu.
  • Configurable Diff Tool: Use any command-line diff tool (WinMerge, KDiff3, Beyond Compare, etc).
  • Cross-Platform Roadmap: Linux and macOS integration coming soon.
