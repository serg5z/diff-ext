Features

  • Context Menu Integration: Adds “Compare”, “Remember”, and MRU-based comparison commands to Windows Explorer’s context menu for files and directories.
  • Compare Files & Folders: Select any two files/folders and compare using your configured diff tool.
  • Most Recently Used (MRU) Support: Remembers previously selected files/folders and allows one-click comparison with the MRU item.
  • Submenu for MRU: See and manage your MRU files directly in the context menu.
  • Configurable Diff Tool: Works with popular diff/merge tools (such as WinMerge KDiff3 Beyond Compare ), configurable by the user.
  • Multi-file manager support: in addition to Windows Explorer, works seamlessly with Explorer++, Q-Dir, and FreeCommander.
  • Multi-Architecture Support: Provides x86, x64, and ARM64 builds in a single installer.
  • Cross-Platform Roadmap: Linux and macOS integration coming soon.
