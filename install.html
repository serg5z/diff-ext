Install

  1. Download: Get the latest release from GitHub or get MSIX package from Windows App Store.
  2. Install: Double-click the diff-ext-installer.exe installer and follow the prompts.
  3. Configure: Set your preferred diff tool in the settings dialog (diff-ext-setup.exe).
  4. Update/Uninstall: Use "Apps & Features" or simply re-run the installer to upgrade.

Requirements:

  • Windows 10/11 (x64, x86, ARM64)
  • External diff tool ( WinMerge, KDiff3, Beyond Compare, etc)
  • Administrator rights to install
